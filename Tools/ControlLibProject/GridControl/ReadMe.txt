========================================================================
    정적 라이브러리 : GridControl 프로젝트 개요
========================================================================

응용 프로그램 마법사에서 이 GridControl 라이브러리를 만들었습니다. 

이 파일에는 GridControl 응용 프로그램을 구성하는 각 파일에 대한
요약 설명이 포함되어 있습니다.


GridControl.vcproj
    응용 프로그램 마법사를 사용하여 생성한 VC++ 프로젝트의 기본 프로젝트 파일입니다. 
    파일을 생성한 Visual C++ 버전에 대한 정보와 응용 프로그램 마법사를 사용하여 선택한
    플랫폼, 구성 및 프로젝트 기능에 대한 정보가 포함되어 있습니다.


/////////////////////////////////////////////////////////////////////////////

StdAfx.h, StdAfx.cpp
    이 파일은 미리 컴파일된 헤더(PCH) 파일인 GridControl.pch와
    미리 컴파일된 형식(PCT) 파일인 StdAfx.obj를 빌드하는 데 사용됩니다.

/////////////////////////////////////////////////////////////////////////////
컴파일러와 링커 스위치가 MFC를 지원할 수 있도록 변경되었습니다. 이 프로젝트에서
MFC 클래스 마법사를 사용하려면 프로젝트에 "resource.h", "GridControl.rc",
그리고 resource.h가 들어 있는 "GridControl.h" 등의
파일을 추가해야 합니다. 정적 라이브러리에 rc 파일을 추가하면 DLL 또는 EXE에는
rc 파일이 하나만 있어야 하는 제약 조건 때문에
어려움을 겪을 수 있습니다. 이 문제를 해결하려면 해당 라이브러리의 .rc 파일을
부모 프로젝트의 .rc 파일에 추가하면 됩니다.
